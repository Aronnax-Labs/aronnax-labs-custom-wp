# Block Preview Environment

Fast development environment for styling and testing blocks outside WordPress.

## Quick Start

```bash
cd dev-preview
npm install
npm run dev
```

Opens at http://localhost:5173 with hot reload.

## Usage

1. Edit `preview.jsx` to import your actual block components
2. Add block styles to `preview.css` or import your block stylesheets
3. Changes appear instantly in browser

## Workflow

- Develop blocks here with instant feedback
- Copy finalized code to `aronnax-labs-custom-blocks/`
- Test in WordPress block editor when ready
